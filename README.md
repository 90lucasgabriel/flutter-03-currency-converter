## 4. Installing and Running
- Clone this repo:
```
git clone git@github.com:90lucasgabriel/flutter-03-currency-converter.git
```

- Access directory:
```
cd flutter-03-currency-converter
```

- Run
```
flutter run
```

- Check browser or connected device;
